# Script de Ventas – PagosYa

Este documento define el proceso completo de ventas para el equipo comercial y de atención.

El objetivo no es vender a la fuerza.

El objetivo es:

- Entender la necesidad del cliente
- Mostrar el valor del sistema
- Generar confianza
- Llevar el cliente a probar el sistema

---

# Regla principal

No hablar demasiado.

No explicar todo de una vez.

Hacer preguntas.

Escuchar.

Responder.

Guiar.

---

# Flujo de venta

El flujo de venta de PagosYa es:

1) Saludo
2) Identificar el negocio
3) Detectar el problema
4) Explicar la solución
5) Mostrar beneficio
6) Ofrecer prueba
7) Cerrar

---

# Paso 1 — Saludo

Mensaje:

Hola 👋  
Gracias por comunicarte con PagosYa.

Somos un sistema que permite confirmar pagos QR automáticamente y tener más control sobre tu negocio.

¿Tu negocio es una tienda, restaurante, farmacia u otro tipo?

---

# Paso 2 — Identificar el negocio

Objetivo:

Entender el tipo de negocio.

Preguntas:

¿A qué se dedica tu negocio?  
¿Tienes empleados?  
¿Aceptas pagos QR?

Nunca asumir.

Siempre preguntar.

---

# Paso 3 — Detectar el problema

Objetivo:

Identificar una necesidad real.

Preguntas:

¿Alguna vez te ha pasado que un cliente mostró un comprobante falso?  

¿Te gustaría tener más control sobre tus ventas?  

¿Quieres evitar errores en caja?

---

# Paso 4 — Explicar la solución

Mensaje:

PagosYa permite confirmar pagos QR automáticamente.

El sistema valida el pago directamente con el banco y te muestra si el pago es real o no.

Además, puedes:

Controlar ventas  
Gestionar inventario  
Registrar empleados  
Ver reportes  
Organizar tu negocio  

Todo en un solo sistema.

---

# Paso 5 — Mostrar beneficio

Nunca hablar solo de funciones.

Hablar de resultados.

Ejemplo:

Con PagosYa puedes:

Evitar pérdidas  
Tener control total  
Ahorrar tiempo  
Trabajar con seguridad  

---

# Paso 6 — Ofrecer prueba

Este es el paso más importante.

Mensaje:

Podemos activar una prueba gratuita para que veas cómo funciona en tu negocio.

La activación toma menos de 1 minuto.

¿Te gustaría probar el sistema?

---

# Paso 7 — Cierre

Opciones de cierre:

Activar prueba  
Agendar demo  
Hablar con asesor  

Mensaje:

Perfecto.

Podemos activarlo ahora mismo o agendar una demostración rápida.

¿Prefieres probar el sistema o ver una demo?

---

# Si el cliente responde PRUEBA

Mensaje:

Excelente.

Vamos a activar tu prueba ahora.

Solo necesito:

Nombre del negocio  
Número de teléfono  
Correo electrónico  

---

# Si el cliente responde DEMO

Mensaje:

Perfecto.

Puedes agendar una demostración en este enlace:

https://calendar.app.google/ymtw35UDCfFv21Sv8

La demostración dura aproximadamente:

15 minutos

---

# Si el cliente responde ASESOR

Mensaje:

Perfecto.

Un asesor se comunicará contigo en breve para ayudarte.

---

# Reglas importantes

Nunca enviar:

Mensajes largos  
Explicaciones técnicas  
Información innecesaria  

Siempre:

Ser claro  
Ser breve  
Ser útil  

---

# Señales de interés

El cliente está interesado cuando:

Hace preguntas  
Pregunta precio  
Pregunta cómo funciona  
Pregunta cómo probar  
Responde rápido  

En ese momento:

Ofrecer prueba inmediatamente.

---

# Señales de desinterés

El cliente no está listo cuando:

Responde muy poco  
Dice "después"  
Dice "no ahora"  

En ese momento:

Mantener la conversación simple  
No presionar  
Programar seguimiento
