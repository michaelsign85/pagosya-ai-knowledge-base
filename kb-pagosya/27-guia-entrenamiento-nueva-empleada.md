# Guía de Entrenamiento – Nueva Empleada – PagosYa

Este documento explica cómo debe entrenarse una nueva persona que trabajará en atención y ventas en PagosYa.

El objetivo es:

- Entender el sistema
- Aprender a atender clientes
- Saber responder correctamente
- Trabajar con seguridad
- Generar ventas

---

# Regla principal

No necesitas saber todo desde el primer día.

Lo importante es:

Aprender  
Practicar  
Preguntar  
Mejorar  

---

# Objetivo del entrenamiento

Al finalizar el entrenamiento, la empleada debe ser capaz de:

Responder mensajes de clientes  
Explicar qué es PagosYa  
Ofrecer la prueba del sistema  
Hacer seguimiento a clientes  
Registrar información correctamente  

---

# Día 5 — Atención independiente

Objetivo:

Atender clientes de forma autónoma.

Actividades:

Responder mensajes  
Ofrecer prueba  
Hacer seguimiento  

---

# Regla de atención

Siempre:

Ser amable  
Ser clara  
Ser breve  
Responder rápido  

Nunca:

Inventar información  
Discutir  
Ignorar mensajes  
Prometer algo que no existe  

---

# Cómo responder correctamente

Usar lenguaje simple.

Ejemplo:

Incorrecto:

Nuestro sistema posee validación automatizada mediante integración bancaria.

Correcto:

El sistema confirma el pago directamente con el banco.

---

# Preguntas que debe hacer al cliente

¿A qué se dedica tu negocio?  

¿Aceptas pagos QR?  

¿Tienes empleados?  

¿Quieres tener más control sobre tus ventas?

---

# Señales de cliente interesado

El cliente:

Pregunta precio  
Pregunta cómo funciona  
Pide información  
Responde rápido  

En ese momento:

Ofrecer prueba.

---

# Señales de cliente no interesado

El cliente:

Responde poco  
Dice después  
No hace preguntas  

En ese momento:

Hacer seguimiento.

---

# Cuándo pedir ayuda

Debe pedir ayuda cuando:

El cliente tiene problema técnico  
El cliente está molesto  
El cliente quiere cancelar  
No sabe qué responder  
Tiene duda
